SECTION 1 — WHAT THE PROJECT IS
────────────────────────────────────────────────────────────────────────────────

The project clusters 17,547 Austrian households by their 2023 energy consumption
patterns, then trains one LightGBM forecasting model per cluster to predict daily
consumption for all of 2024.

The assignment requires two things from you:
  1. Evaluate the cluster-based models (already done — results exist)
  2. Train and evaluate a GLOBAL baseline model (not yet compared properly)

The global model already exists as a saved .pkl file. You just need to score it
against the 2024 ground truth and compare its MAE to the cluster models.
This single comparison number is the headline result of the entire project.

SECTION 4 — FILES FOR EVALUATION 
────────────────────────────────────────────────────────────────────────────────

  results/predictions/predictions_2024.csv
    The main prediction file from NB04. 6,422,202 rows.
    Columns: household_id, date, predicted, actual, cluster, cluster_name, model_used
    The 'actual' column is already joined in — it comes from sample_24.csv.
    This is what you use to evaluate cluster model performance.

  results/predictions/predictions_2024_probabilistic.csv
    Same but with prediction intervals from NB05. 6,422,202 rows.
    Columns: household_id, date, lower, predicted, upper, actual, cluster,
             cluster_name, model_used, interval_width

  results/models/lgbm_global.pkl
    The trained global LightGBM model. Trained on all 16,494 regular households
    using 2023 data. This is the dataset-level baseline the assignment requires.
    You need to run this model on 2024 features and compute its MAE.

  results/models/lgbm_cluster_0.pkl through lgbm_cluster_3.pkl
    The four cluster-specific models. Already evaluated in NB04.

  results/diagnostics/cluster_assignments.csv
    Maps every household_id to its cluster label (-3, -2, -1, 0, 1, 2, or 3).
    Negative labels = degenerate households. Used by NB04 and needed if you
    re-run any feature engineering.

  results/metrics/forecast_metrics_per_cluster.csv
    Per-cluster MAE, RMSE, MAPE, R² from NB04. Already computed.
    cluster, cluster_name, n_households, n_rows, mae, rmse, mape, r2

  results/metrics/forecast_metrics_overall.csv
    Single row: overall MAE = 2.1749, RMSE = 4.9824, R² = 0.828.

  results/metrics/forecast_metrics_probabilistic.csv
    Per-cluster metrics from NB05 including coverage and Winkler score.

  results/metrics/household_coverage.csv
    Per-household coverage, PIAW, and MAE from NB05. 17,547 rows.
